09 — Figma Setup Checklist

1. Install fonts

  • Bebas Neue — Google Fonts (free)
  • DM Sans — Google Fonts (free), weights 300 · 400 · 500
  • DM Sans Italic 300 (hero descriptor optional)

2. Local variables / colour styles

  • 8 colour tokens from Section 01
  • Group under "Insignia/" prefix
  • Set Dark as page fill in frames

3. Text styles

  • 8 text styles from Section 02
  • Group under "Insignia/" prefix
  • Note: Bebas Neue has only 1 weight

4. Frame sizes

  • Desktop: 1440 × 900 (primary)
  • Tablet: 900 × 1024 (breakpoint)
  • Mobile: 390 × 844
  • Max content width: 1300px centred

5. Grid setup (Figma layout grid)

  • Columns: 12 · Gutter: 24px · Margin: 48px
  • Colour: accent gold at 10% fill

6. Components to build

  • Nav bar (default + scrolled states)
  • Project card (default + hover)
  • Family tag badge
  • Hero descriptor block
  • Hero meta stats row
